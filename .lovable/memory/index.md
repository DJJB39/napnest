# Project Memory

## Core
Dark navy #0F172A theme, lavender #A78BFA accent. Plus Jakarta Sans headings, Inter body, JetBrains Mono timers.
Baby sleep tracker app called "NapNest" (rebranded from DreamLog). Mobile-first, bottom nav, dark mode default.
Supabase with RLS via is_family_member() security definer. Realtime on sleep_entries.
NHS-aligned wake windows and sleep guidance. AI reviews via Gemini 2.5 Flash.
